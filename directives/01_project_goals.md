# 1. Mục tiêu cốt lõi (Core Goals)
- **Sản phẩm:** Xây dựng Landing Page giới thiệu mang tên "Guruji ruma web".
- **Tính chất:** Thiên về UI/UX mượt mà, áp dụng Animation bắt mắt, mang lại ấn tượng "Wow" từ cái nhìn đầu tiên. Đảm bảo tối ưu tốc độ tải trang và tuân thủ các chuẩn mực SEO.
- **Tiêu chí Hoàn thành (DONE):** Giao diện chạy mượt 100% không lỗi trên cả Desktop và Mobile (Responsive tuyệt đối). Hình ảnh/Assets được tối ưu hóa.

# 2. In/Out of Scope (Phạm vi)
- **Trong vòng vi (In Scope):** Lên toàn bộ Layout Frontend, tập trung hoàn thành UI/UX xuất sắc ở Giai đoạn 1. Tích hợp dữ liệu dạng tĩnh (từ file JSON hoặc hardcode) để thể hiện giao diện.
- **Ngoài phạm vi (Out of Scope):** Không xây dựng Database, API thực, Logic chức năng nâng cao (như Authentication, Payment, Backend). Các phần này sẽ dời qua Phase 2 nếu cần thiết.
